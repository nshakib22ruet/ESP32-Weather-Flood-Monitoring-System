# ESP32 IoT Weather & Flood Monitoring System

An intelligent ESP32-based IoT monitoring system that collects real-time temperature, humidity, rainfall, and water-level data and visualizes them through an interactive web dashboard. The system utilizes a Quadratic Regression Machine Learning model to predict environmental conditions 2 minutes ahead, providing early flood-risk insights and trend forecasting.

## Key Features

* Real-time monitoring of temperature, humidity, rainfall, and water level
* Interactive HTML graphs for live sensor data visualization
* 2-minute ahead prediction using Quadratic Regression
* Real-time comparison of measured and predicted values
* SH1106 OLED display for local status monitoring
* Wireless data transmission between ESP32 and web dashboard
* Historical trend analysis and flood-risk indication
* Lightweight and low-cost IoT-based smart monitoring solution

## Technologies Used

* ESP32
* Arduino IDE
